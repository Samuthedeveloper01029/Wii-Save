# Wii-Save
Is your Wii full of save files? Is it running out of memory? Is your SD card unable to read the Wii saves you transferred? Do you have an empty USB? Well, Wii Save is the app for you. It can create a folder on your USB or Hard Disk with your save files and you can use them instead of Wii ones. Based on the new technology SNANDE (created by me!!)
